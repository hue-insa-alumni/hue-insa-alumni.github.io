# Hué-INSA Alumni Wiki

Cổng thông tin chung của hội Hué-INSA Alumni, xây dựng bằng [Zensical](https://github.com/zensical/zensical).

## Dev

```bash
make install-uv   # cài uv (nếu chưa có)
make setup        # cài Zensical
make run          # dev server tại http://localhost:8000
```

Xem hướng dẫn đầy đủ tại [Hướng dẫn sử dụng](contents/huong-dan.md).
